Black Scholes Merton Option Pricing Calculator

Usage

Click on the above Calculator tab to redirect to the calculator. From there, fill in the data into the appropriate fields and click on the calculate button. You should be able to receive an estimated option price based on the data provided

About

This web application is intended to refine my skills as a developer and financial engineer. Using prior knowledge from course materials, I have created a quick and simple solution to price European options using the Black Scholes Merton differential equation
